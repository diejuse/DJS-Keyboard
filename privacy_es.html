Política de Privacidad — DJS Keyboard

Última actualización: 12 de diciembre de 2025

DJS Keyboard (“la aplicación”) es un teclado para Android desarrollado por DJS. Esta aplicación respeta completamente la privacidad del usuario y no recopila, almacena ni comparte ningún dato personal.

1. Datos que recoge la aplicación

DJS Keyboard NO recopila ningún tipo de información personal, incluyendo:

  • Pulsaciones o texto escrito
  • Nombres de usuario, contraseñas u otra información sensible
  • Contactos, mensajes, fotos o archivos
  • Datos de ubicación
  • Información del dispositivo
  • Cualquier dato identificable

Todo el procesamiento que realiza el teclado se ejecuta únicamente en tu dispositivo.

2. Permisos utilizados

La aplicación utiliza únicamente los permisos necesarios para su funcionamiento:

  • Input Method Service: necesario para que el teclado funcione en Android. No se usa para recopilar datos del usuario.

3. Servicios de terceros

DJS Keyboard no utiliza:

  • Analíticas
  • Publicidad
  • SDKs externos
  • Servicios que recopilen o procesen información personal

4. Compras integradas

La aplicación puede incluir compras integradas para desbloquear funciones premium. El procesamiento del pago es gestionado exclusivamente por Google Play, que ofrece sus propios sistemas de seguridad y privacidad.

La aplicación no recibe ni almacena información de pago, incluyendo:

  • Datos de tarjetas
  • Información bancaria
  • Detalles de facturación
  • Identificadores de transacciones

5. Seguridad

Como la aplicación no recopila datos personales, no existe riesgo de filtración o exposición de datos sensibles a través de esta app.

6. Menores

La aplicación puede ser utilizada por usuarios de todas las edades. DJS Keyboard no recopila información de menores ni de adultos.

7. Cambios en esta política

Podemos modificar esta Política de Privacidad ocasionalmente. La versión más reciente estará siempre disponible en este enlace.
